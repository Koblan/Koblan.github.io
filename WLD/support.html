ВВЕДЕНИЕ

Суть работ по поддержке интернет-ресурсов заключается в поддержании его в работоспособном состоянии с актуальной информацией. Так же сюда входят мероприятия по резервированию сайта, проверке его на вирусы.

Для разных типов интернет-сайтов набор услуг разнится. Например для интернет-магазинов актуальны услуги по обслуживанию каталога продукции, для промо-сайтов - обновление новостного содержимого.

Отдельная услуга - повышение качество кода готовых ресурсов для поискового продвижения.

УСЛУГИ ПОДДЕРЖКИ

Обновление содержимого

Обновление новостей, блогов, статей, отзывов. Содержимое может предоставляться или разрабатываться на заказ в соответствии с заданием SEO-специалиста.

Поддержка каталога

Добавление новых позиций, удаление старых, синхронизация цен и прочих характеристик. Выгрузка данных каталога. Синхронизация со складским ПО.

< подробнее >

SEO-поддержка

Поддержка работ по поисковому продвижению: оптимизация кода, рамещение мета-данных, уменьшение количества JS, повышение значений теста Google PageSpeed Insight.

< подробнее >

Техническое обслуживание

Создание регулярных резервных копий, контроль доступности ресурса, контроль сроков оплаты хостинга и доменных имён. Проверка на вирусы. Настройка почты.

< подробнее >

Дизайн-поддержка

Разработка внешнего вида элементов: баннеров, разделов, элементов интерфейса. Работы выполняются на постоянной основе прикреплённым к заказчику дизайнером.

Управление баннерами

Создание, удаление, замена баннеров. Замена изображений, текста, ссылок. Изменение места вывода баннеров. Систематическая аналитика переходов.

ПОРЯДОК СОТРУДНИЧЕСТВА

Работы начинаются с формирования заказчиком технического задания с описанием необходимых работ, а так же сроков их исполнения. Исключением являются работы по поддержке в рамках фиксированного тарифного плана.
